# Array Fields

React Any Shape Form provides powerful support for dynamic array fields, enabling you to build complex forms with repeating sections, lists, and dynamic collections.

Array field support is an **optional addon** — import `withArrayFields` only when you need it, keeping the core bundle lean.

## Setup

```tsx
import { useForm } from 'react-any-shape-form';
import { withArrayFields } from 'react-any-shape-form/addons/array';

const MyForm = useForm(
  { tags: [] as string[] },
  withArrayFields(),
);
```

## Overview

Array fields support:
- **Dynamic add/remove** - Users can add and remove items at runtime
- **Full array operations** - append, prepend, remove, update, move
- **Type safety** - Full TypeScript support for array elements
- **Validation** - Both array-level and item-level validation
- **Performance** - Efficient re-renders with proper key management

## Basic Usage

### Using Form.ArrayItem

The `Form.ArrayItem` component provides a declarative way to work with arrays.
The render prop receives the full array API:

```tsx
import { withArrayFields } from 'react-any-shape-form/addons/array';

const MyForm = useForm(
  { tags: [] as string[] },
  withArrayFields(),
);

<MyForm.ArrayItem name="tags">
  {({ value, append, remove, update }) => (
    <div>
      {value.map((tag, index) => (
        <div key={index}>
          <input
            value={tag}
            onChange={(e) => update(index, e.target.value)}
            placeholder="Tag name"
          />
          <button onClick={() => remove(index)}>Remove</button>
        </div>
      ))}
      <button onClick={() => append("")}>Add Tag</button>
    </div>
  )}
</MyForm.ArrayItem>
```

### Using useArrayField Hook

For more flexibility, use the `useArrayField` hook directly:

```tsx
import { withArrayFields } from 'react-any-shape-form/addons/array';

const MyForm = useForm(
  { emails: [] as string[] },
  withArrayFields(),
);

function EmailList() {
  const { value, append, remove, update } = MyForm.useArrayField("emails");

  return (
    <div>
      {value.map((email, index) => (
        <div key={index}>
          <input
            value={email}
            onChange={(e) => update(index, e.target.value)}
            type="email"
          />
          <button onClick={() => remove(index)}>Delete</button>
        </div>
      ))}
      <button onClick={() => append("")}>Add Email</button>
    </div>
  );
}
```

## Array Operations

### append

Add an item to the end of the array:

```tsx
const { append } = MyForm.useArrayField("items");

// Simple values
append("");
append(0);
append(true);

// Objects
append({ name: "", email: "" });

// With default values
<button onClick={() => append({ name: "New Item", quantity: 1 })}>
  Add Item
</button>
```

### prepend

Add an item to the beginning of the array:

```tsx
const { prepend } = MyForm.useArrayField("notifications");

prepend({
  id: Date.now(),
  message: "New notification",
  read: false,
});
```

### remove

Remove an item by index:

```tsx
const { value, remove } = MyForm.useArrayField("items");

// Remove specific item
<button onClick={() => remove(index)}>Delete</button>

// Remove first item
<button onClick={() => remove(0)}>Delete First</button>

// Remove last item
<button onClick={() => remove(value.length - 1)}>Delete Last</button>
```

### update

Update an item by index. Accepts a new value or an updater function:

```tsx
const { value, update } = MyForm.useArrayField("todos");

// Update string value
<input
  value={value[index]}
  onChange={(e) => update(index, e.target.value)}
/>

// Update object property via updater function
<input
  value={value[index].title}
  onChange={(e) => update(index, (prev) => ({ ...prev, title: e.target.value }))}
/>
```

### move

Move an item from one index to another:

```tsx
const { value, move } = MyForm.useArrayField("items");

// Move item up
<button onClick={() => move(index, index - 1)} disabled={index === 0}>
  Move Up
</button>

// Move item down
<button onClick={() => move(index, index + 1)} disabled={index === value.length - 1}>
  Move Down
</button>

// Move to top
<button onClick={() => move(index, 0)}>
  Move to Top
</button>
```

## Working with Different Data Types

### String Arrays

```tsx
const MyForm = useForm(
  { tags: [] as string[] },
  withArrayFields(),
);

function TagInput() {
  const { value, append, remove, update } = MyForm.useArrayField("tags");

  return (
    <div>
      {value.map((tag, index) => (
        <div key={index}>
          <input
            value={tag}
            onChange={(e) => update(index, e.target.value)}
          />
          <button onClick={() => remove(index)}>×</button>
        </div>
      ))}
      <button onClick={() => append("")}>Add Tag</button>
    </div>
  );
}
```

### Number Arrays

```tsx
const MyForm = useForm(

function ScoreList() {

  return (

      {value.map((score, index) => (

    </div>

```

### Object Arrays

```tsx
interface Contact {

const MyForm = useForm(

          />

            placeholder="Email"
            type="email"
            value={contact.email}
            onChange={(e) => update(index, (prev) => ({ ...prev, email: e.target.value }))}

            placeholder="Phone"
            value={contact.phone}

```

### Nested Arrays

```tsx
interface TodoList {
  title: string;
  items: string[];
}

const MyForm = useForm(
  { todoLists: [] as TodoList[] },
  withArrayFields(),
);

function NestedTodoLists() {
  const { value: lists, append: appendList, update: updateList } =
    MyForm.useArrayField("todoLists");

  return (
    <div>
      {lists.map((list, listIndex) => (
        <div key={listIndex}>
          <h3>{list.title}</h3>
          {list.items.map((item, itemIndex) => (
            <div key={itemIndex}>
              <input
                value={item}
                onChange={(e) => {
                  const newItems = [...list.items];
                  newItems[itemIndex] = e.target.value;
                  updateList(listIndex, (prev) => ({ ...prev, items: newItems }));
                }}
              />
            </div>
          ))}
          <button onClick={() => {
            updateList(listIndex, (prev) => ({
              ...prev,
              items: [...prev.items, ""],
            }));
          }}>
            Add Item to List
          </button>
        </div>
      ))}
      <button onClick={() => appendList({ title: "New List", items: [] })}>
        Add List
      </button>
    </div>
  );
}
```

## Validation

### Array-Level Validation

Validate the array as a whole (length, required, etc.):

```tsx
<MyForm.ArrayItem
  name="emails"
  rules={[
    { required: true, message: "At least one email is required" },
    { type: "array", min: 1, message: "Add at least one email" },
    { type: "array", max: 5, message: "Maximum 5 emails allowed" },
  ]}
>
  {({ value, append, remove }) => (
    // render fields
  )}
</MyForm.ArrayItem>
```

### Item-Level Validation

Validate each array element individually. Use the `items` prop (instead of `value`) to get per-item error and validation status:

```tsx
<MyForm.ArrayItem
  name="emails"
  itemRules={[
    { required: true, message: "Email is required" },
    { type: "email", message: "Invalid email format" },
  ]}
>
  {({ items, itemErrors }) => (
    <div>
      {items.map(({ value: email, index, onChange, errors, validationStatus }) => (
        <div key={index}>
          <input
            value={email}
            onChange={(e) => onChange(e.target.value)}
            className={errors.length > 0 ? 'error' : ''}
          />
          {errors.length > 0 && (
            <span className="error">
              {errors.map(e => e.errorText).join(', ')}
            </span>
          )}
        </div>
      ))}
    </div>
  )}
</MyForm.ArrayItem>
```

### Combined Validation

Pass array-level and item-level rules to `useArrayField`:

```tsx
const { value, append, remove, update, itemErrors } = MyForm.useArrayField(
  "tags",
  // Array-level rules (second argument)
  [
    { type: "array", min: 1, max: 10, message: "Between 1 and 10 tags" },
  ],
  // Item-level rules (third argument)
  [
    { required: true, message: "Tag cannot be empty" },
    { type: "string", min: 2, max: 20, message: "Tag must be 2-20 characters" },
  ],
);
```

For detailed validation documentation, see [Array Item Validation](/docs/documentation-array-item-validation--docs).

## Common Patterns

### Todo List

```tsx
interface Todo {
  id: number;
  text: string;
  completed: boolean;
}

const MyForm = useForm(
  { todos: [] as Todo[] },
  withArrayFields(),
);

function TodoList() {
  const { value, append, update, remove } = MyForm.useArrayField("todos");

  return (
    <div>
      {value.map((todo, index) => (
        <div key={todo.id}>
          <input
            type="checkbox"
            checked={todo.completed}
            onChange={(e) =>
              update(index, (prev) => ({ ...prev, completed: e.target.checked }))
            }
          />
          <input
            value={todo.text}
            onChange={(e) =>
              update(index, (prev) => ({ ...prev, text: e.target.value }))
            }
            style={{ textDecoration: todo.completed ? 'line-through' : 'none' }}
          />
          <button onClick={() => remove(index)}>Delete</button>
        </div>
      ))}
      <button onClick={() => append({ id: Date.now(), text: "", completed: false })}>
        Add Todo
      </button>
    </div>
  );
}
```

### Tag Input with Enter Key

```tsx
const MyForm = useForm(
  { tags: [] as string[] },
  withArrayFields(),
);

function TagInput() {
  const { value, append, remove } = MyForm.useArrayField("tags");
  const [inputValue, setInputValue] = useState("");

  const handleKeyDown = (e: React.KeyboardEvent) => {
    if (e.key === "Enter" && inputValue.trim()) {
      e.preventDefault();
      append(inputValue.trim());
      setInputValue("");
    }
  };

  return (
    <div>
      <div className="tags">
        {value.map((tag, index) => (
          <span key={index} className="tag">
            {tag}
            <button onClick={() => remove(index)}>×</button>
          </span>
        ))}
      </div>
      <input
        value={inputValue}
        onChange={(e) => setInputValue(e.target.value)}
        onKeyDown={handleKeyDown}
        placeholder="Type and press Enter"
      />
    </div>
  );
}
```

### Sortable List with Drag and Drop

```tsx
const MyForm = useForm(
  { items: ["Item 1", "Item 2", "Item 3"] },
  withArrayFields(),
);

function SortableList() {
  const { value, move } = MyForm.useArrayField("items");
  const [draggedIndex, setDraggedIndex] = useState<number | null>(null);

  const handleDragOver = (e: React.DragEvent, index: number) => {
    e.preventDefault();
    if (draggedIndex !== null && draggedIndex !== index) {
      move(draggedIndex, index);
      setDraggedIndex(index);
    }
  };

  return (
    <div>
      {value.map((item, index) => (
        <div
          key={index}
          draggable
          onDragStart={() => setDraggedIndex(index)}
          onDragOver={(e) => handleDragOver(e, index)}
          className="draggable-item"
        >
          {item}
        </div>
      ))}
    </div>
  );
}
```

### Shopping Cart

```tsx
interface CartItem {
  id: string;
  name: string;
  price: number;
  quantity: number;
}

const MyForm = useForm(
  { cart: [] as CartItem[] },
  withArrayFields(),
);

function ShoppingCart() {
  const { value, update, remove } = MyForm.useArrayField("cart");

  const total = value.reduce((sum, item) => sum + item.price * item.quantity, 0);

  return (
    <div>
      {value.map((item, index) => (
        <div key={item.id} className="cart-item">
          <div>{item.name}</div>
          <div>\${item.price.toFixed(2)}</div>
          <input
            type="number"
            min="1"
            value={item.quantity}
            onChange={(e) =>
              update(index, (prev) => ({ ...prev, quantity: Number(e.target.value) }))
            }
          />
          <div>\${(item.price * item.quantity).toFixed(2)}</div>
          <button onClick={() => remove(index)}>Remove</button>
        </div>
      ))}
      <div className="total">Total: \${total.toFixed(2)}</div>
    </div>
  );
}
```

## Performance Optimization

### Use Stable Keys

Always use stable keys for array items. If your items don't have IDs, add them:

```tsx
// ✗ Bad — index as key causes issues when reordering
{value.map((item, index) => (
  <div key={index}>{item}</div>
))}

// ✓ Good — stable ID from the item itself
interface Item {
  id: string;
  value: string;
}

const { value } = MyForm.useArrayField("items");

{value.map((item) => (
  <div key={item.id}>{item.value}</div>
))}
```

### Memoize Array Item Components

Use `React.memo` for array item components to avoid unnecessary re-renders:

```tsx
const TodoItem = React.memo(({
  todo,
  index,
  onUpdate,
  onRemove,
}: {
  todo: Todo;
  index: number;
  onUpdate: (index: number, updater: (prev: Todo) => Todo) => void;
  onRemove: (index: number) => void;
}) => (
  <div>
    <input
      value={todo.text}
      onChange={(e) => onUpdate(index, (prev) => ({ ...prev, text: e.target.value }))}
    />
    <button onClick={() => onRemove(index)}>Delete</button>
  </div>
));

function TodoList() {
  const { value, update, remove } = MyForm.useArrayField("todos");

  return (
    <div>
      {value.map((todo, index) => (
        <TodoItem
          key={todo.id}
          todo={todo}
          index={index}
          onUpdate={update}
          onRemove={remove}
        />
      ))}
    </div>
  );
}
```

## Type Safety

The library provides full TypeScript support for array fields:

```tsx
const MyForm = useForm(
  {
    strings: [] as string[],
    numbers: [] as number[],
    objects: [] as { name: string; age: number }[],
  },
  withArrayFields(),
);

// TypeScript knows the element types
const { value: strings } = MyForm.useArrayField("strings");
// strings: string[]

const { value: numbers } = MyForm.useArrayField("numbers");
// numbers: number[]

const { value: objects } = MyForm.useArrayField("objects");
// objects: { name: string; age: number }[]

// Only array fields are allowed — TypeScript error for non-array fields:
const { value } = MyForm.useArrayField("nonArrayField"); // TS error!

// Without withArrayFields(), ArrayItem and useArrayField don't exist on the type:
const BasicForm = useForm({ name: "" });
BasicForm.ArrayItem; // TS error — Property 'ArrayItem' does not exist
```

## FormApi Methods

Access array fields programmatically (no addon required — these are on `FormApi` directly):

```tsx
// Get array field value
const tags = MyForm.formApi.getFieldValue("tags");

// Set entire array
MyForm.formApi.setFieldValue("tags", ["tag1", "tag2", "tag3"]);

// Append to array
const current = MyForm.formApi.getFieldValue("tags");
MyForm.formApi.setFieldValue("tags", [...current, "newTag"]);

// Validate array
await MyForm.formApi.validateFields(["tags"]);
```

## Best Practices

### 1. Provide Default Values

Always append with complete default values:

```tsx
// ✓ Good
append({ name: "", email: "", phone: "" });

// ✗ Bad — incomplete object
append({ name: "" }); // Missing email and phone
```

### 2. Use Validation

Always validate array inputs, especially for user-generated content:

```tsx
const { value, itemErrors } = MyForm.useArrayField(
  "emails",
  [{ type: "array", min: 1, message: "Add at least one email" }],
  [{ type: "email", message: "Invalid email" }],
);
```

### 3. Handle Empty States

Show helpful messages when arrays are empty:

```tsx
{value.length === 0 ? (
  <div className="empty-state">
    <p>No items yet. Click "Add Item" to get started.</p>
  </div>
) : (
  value.map((item, index) => (
    // render item
  ))
)}
```

### 4. Confirm Destructive Actions

Ask for confirmation before removing items:

```tsx
const handleRemove = (index: number) => {
  if (confirm("Are you sure you want to remove this item?")) {
    remove(index);
  }
};
```

### 5. Limit Array Size

Prevent users from adding too many items:

```tsx
const MAX_ITEMS = 10;

<button
  onClick={() => append("")}
  disabled={value.length >= MAX_ITEMS}
>
  Add Item {value.length >= MAX_ITEMS && "(Maximum reached)"}
</button>
```
